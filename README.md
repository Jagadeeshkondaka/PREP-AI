# 🚀 PrepAI – AI-Powered Career Assistant

PrepAI is a full-stack AI-powered web application designed to help job seekers improve their resumes, practice interviews, and access personalized learning resources — all in one platform.

---

## 🌟 Features

- 📄 **Resume Analyzer**
  - Upload resume (PDF)
  - Get ATS score and AI-based feedback
  - Improve resume with actionable suggestions

- 💬 **AI Interview Preparation**
  - Generate role-based interview questions
  - Submit answers and receive AI feedback
  - Improve communication and confidence

- 📚 **Smart Resource Finder**
  - Get personalized learning resources using AI
  - Tailored suggestions based on user input

- 🔐 **Authentication System**
  - Secure login/signup
  - Protected routes (only logged-in users can access features)

---
